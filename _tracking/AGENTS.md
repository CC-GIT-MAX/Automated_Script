# AGENTS.md -- 07_Tracking

This folder tracks the **state of the automation system**: pending work,
future ideas, and historical changes.

## Files

- `TODO.md` -- tasks that need to be done (priority-ordered)
- `IMPROVEMENTS.md` -- ideas for future enhancements (not committed)
- `CHANGELOG.md` -- dated log of every change to this repository

## Conventions

- When you START a task, add a `[WIP]` tag next to it in TODO.md
- When you FINISH a task, remove it from TODO.md and add a row to CHANGELOG.md
- IDEAS go in IMPROVEMENTS.md until they''re committed to a release plan
- CHANGELOG.md uses the format: `## YYYY-MM-DD -- <summary>` followed by bullets
